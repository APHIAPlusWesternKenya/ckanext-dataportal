    ILRI's Datasets Portal API

    ILRI's Datasets Portal can provide access to third party systems using an API. The result of an API call is in JSON format that can be easily read by computer programs. Below are examples of how to get different kinds of data. More information on the API can be found in CKAN's API documentation.

    • List of datasets.

      {{host}}/portal/api/3/action/package_list . The result of this call is a list dataset names that can be used in other API calls.

    • Details about a dataset

      {{host}}/portal/api/3/action/package_show?id=MyDatasetName. Change the ID to any dataset name

    • List of AGROVOC Tags

      {{host}}/portal/api/3/action/tag_list.

    • Dataset Metadata in DDI XML Format

      {{host}}/portal/dataset/MyDatasetName/metadataddi.
